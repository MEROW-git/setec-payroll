# Human Resource Payroll

Siegecode HRM is a modern Human Resource Management and payroll interface for tracking employees, attendance, leave, payroll, performance, and business operations from one dashboard.

## Project Structure

```text
Human-Resource-Payroll/
├── backend/
└── frontend/
    ├── src/
    ├── dist/
    ├── package.json
    └── vite.config.ts
```

- `frontend/` contains the React, TypeScript, Vite, and Tailwind CSS application.
- `backend/` is reserved for backend API code.

## Frontend Setup

Open a terminal from the project root and run:

```bash
cd frontend
npm install
npm run dev
```

Vite will print a local URL, usually:

```text
http://localhost:5173
```

## Frontend Build

To create a production build:

```bash
cd frontend
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Features

- Employee dashboard and workforce overview
- Attendance and leave tracking
- Payroll and compensation overview
- Performance and KPI insights
- Sidebar navigation and operational widgets
- Responsive React interface with Tailwind CSS
